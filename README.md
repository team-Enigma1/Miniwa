# Miniwa

## 概要
Miniwa は、**初めて家庭菜園に挑戦する人**を対象にした、教科書のようなモバイルアプリケーションです。  
ユーザーの状況や植物の成長段階に合わせて家庭菜園をサポートし、アプリを開けば **「今、何をすればいいのか」** が分かります。

ターゲットは、これからベランダ家庭菜園を始めたい人です。  
ユーザーごとに植物の成長記録を残すことができ、自分専用の家庭菜園ガイドとして活用できます。

Webサイトや書籍でも家庭菜園の情報は得られますが、情報が抽象的だったり、1から100まで一度に書かれていることが多く、初心者には難しいと感じることがあります。  
Miniwa では、スマートフォンを開くだけで **「自分だけの教科書が開く」** ような体験を目指して制作しました。
